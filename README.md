# Google Voice Clone - Dialer Application

A modern, responsive web-based Dialer application inspired by Google Voice, built with React, Vite, and Tailwind CSS.

## Features

- **Google-inspired UI**: Clean, minimal, and premium design.
- **Responsive Dialer**: Works on mobile and desktop.
- **Audio Feedback**: Subtle tones when pressing dialpad buttons.
- **Dark Mode**: Seamless toggle between light and dark themes.
- **Call History**: List of recent calls with dummy data.
- **Animations**: Smooth transitions using Framer Motion.
- **Simulated Calling**: Mock "Calling" and "Connected" states.

## Tech Stack

- **React.js** (Functional Components, Hooks)
- **Vite** (Build Tool)
- **Tailwind CSS** (Styling)
- **Lucide React** (Icons)
- **Framer Motion** (Animations)
- **TypeScript** (Type safety)

## Getting Started

1. **Install Dependencies**:
   ```bash
   npm install
   ```

2. **Run Development Server**:
   ```bash
   npm run dev
   ```

3. **Build for Production**:
   ```bash
   npm run build
   ```

## Folder Structure

- `src/components/`: Modular React components (Sidebar, Dialpad, etc.)
- `src/utils/`: Mock data and helpers.
- `src/types.ts`: TypeScript interfaces.
- `src/index.css`: Global styles and Tailwind configuration.
